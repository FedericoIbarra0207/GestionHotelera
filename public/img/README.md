# Carpeta de Imágenes - DVT Software

## ⚠️ INSTRUCCIONES

Coloca aquí tu archivo de logo:

**Archivo requerido:**
- `dvt-logo.ico` - El logo que creaste (formato ICO o PNG)

## 📋 Ubicación

```
hoteleria/
├── public/
│   ├── img/
│   │   └── dvt-logo.ico  ← PEGA AQUÍ TU LOGO
│   ├── index.html
│   ├── styles.css
│   └── app.js
```

## 🎨 Especificaciones

- **Formato:** .ico (recomendado) o .png (si es PNG, cambiar la ruta en index.html)
- **Tamaño:** 40x40 pixels (se escala automáticamente)
- **Nombre del archivo:** `dvt-logo.ico` (exactamente este nombre)

## ✅ Listo

Una vez que copies el archivo aquí, el logo aparecerá automáticamente en el navbar morado, junto al texto "DVT".
